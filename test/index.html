My first elevator pitch is a Wordle Suggestion application.
This is helpful to people who would like a 5 letter word suggested to them.
I would call this Wordle Suggestions.
This is an application that would suggest a 5 letter word to the user, depending on what is entered.
Unlike a dictionary, this product would change the suggestion depending on the input.

My second elevator pitch is a PDF reader with custom color choices.
This is for people who read .pdfs.
This is helpful to people who would like to read pdfs in a certain color scale, dark mode or high contrast.
I would call this the PDF Color Changer.
This allows the user to set the exact colors of text and background.
Unlike Dark-reader, this product allows the colors to be chosen.

My third elevator pitch is a scaling visualization application.
This is for people who would like to visualize the size of a measurement.
This is helpful to people who would like to see an object in relation to a measurement.
I would call this For Scale.
This allows the user to enter a measurement and see an object that is that size.
Unlike a measurement, this product allows the user to visualize the measurement with an object.
